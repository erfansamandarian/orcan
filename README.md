# installation 

## venv

```
git clone https://github.com/erfansamandarian/orcan
```

```
cd orcan
```

```
python -m venv venv
```

```
pip install .
```

## pypi

```
pip install orcan
```

or, if you need to

```
pip install orcan --break-system-packages
```

# usage

```
orcan --url {address} --{option}
```

## options

| **argument** | **description**      |
|--------------|----------------------|
| --domain     | print subdomains     |
| --who        | print whois response |
